# WebApp-Kata

WebApp-Kata same as Code Kata, only one thing different is built on the Web platform.

## Prepare

Install MongoDB and turn on the service.

## Install

Install neccesary libraries.

```
npm install
```

## How to Run

```
npm start
```
## How to Test

```
npm install -g mocha

npm start
```

## JSHint

```
grunt
```
